# UI Polish: Beta Badge, Icon Migration, Layout Updates

*March 5-6, 2026*

---

## Summary

A set of visual refinements across the app: beta badge branding, migration from emoji/Lucide icons to Heroicons, sidebar layout improvements, and minor fixes.

## Changes

### Beta Badge

Added a "Beta" badge next to the Lingle logo on the `/get-started` and `/onboarding` pages. Styled with design system tokens: `text-[9px] font-semibold tracking-wide uppercase bg-bg-hover text-text-secondary border border-border-strong`. Sets user expectations that the product is in early access.

### Icon Migration (Lucide → Heroicons)

Migrated the entire app from `lucide-react` emoji-style icons to `@heroicons/react/24/outline`. This affects:
- Sidebar navigation items (all changed from emoji strings to React icon components)
- Conversation view suggestion icons
- Voice components
- Progress pages

Added `@heroicons/react` as a dependency.

### Sidebar Layout

- `UserFooter` now accepts a `collapsed` prop for future responsive sidebar behavior
- Sign-out redirects to `/` instead of `/sign-in`
- Removed commented-out nav sections (Learn, Review)
- Added "Plan" nav item under the Track section linking to `/upgrade`

### Session Analysis Caching

Added `cachedAnalysis Json?` field to `ConversationSession`. Post-session analysis results are now cached in the DB so they don't need to be re-computed when revisiting a session.

### Misc Fixes

- Onboarding redirect logic fixed: now checks `user.onboardingCompleted` via `/api/user/me` instead of checking for profile existence
- OAuth redirect simplified (removed hardcoded `?next=/onboarding`)
- DB migration for new Subscription and DailyUsage tables
